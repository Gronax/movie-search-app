## 🚀 Live Demo
[movie-search-app-livid-five.vercel.app](https://movie-search-app-livid-five.vercel.app/)

## 📦️ Usage
```bash
git clone https://github.com/Gronax/movie-search-app.git
cd movie-search-app
npm install
npm run dev
```
## 📦️ Environment
```bash
VITE_API_KEY=YOUR_KEY
VITE_API_URL=http://www.omdbapi.com
```
